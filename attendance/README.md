# 私人考勤小助手

**希望能为打卡解放沟通成本！**

*当然，主要是因为我们公司没有打卡系统*

## 打卡仅需两步>>
1. 点击日期
2. 选择到岗情况

## 附加功能：
**到岗情况统计**——管理者一目了然无需额外沟通


## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
